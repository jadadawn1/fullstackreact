# Fullstack Product Store

This is a fullstack web application using React and Supabase. It features:

- User authentication
- Admin dashboard
- User profile management
- Products list (fetched from Supabase)

## Getting Started

1. Clone this repository:
   ```bash
   git clone <your-repo-url>
   ```
2. Install dependencies:
   ```bash
   npm install
   ```
3. Set up your Supabase project and update `supabaseClient.js` with your credentials.
4. Start the development server:
   ```bash
   npm start
   ```

## Folder Structure
- `src/` - React source files
- `public/` - Static files

## Features
- Authentication (login/logout)
- Admin-only access to admin dashboard
- Profile page for users
- Products list fetched from Supabase
